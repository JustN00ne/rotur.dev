Features

Friends

Rotur has a friends system out of the box, just authenticate into an account and you can add or remove friends, and send requests

Networking

Rotur uses a modified version of cloudlink to send and recieve packets with much more user data that can be useful to the reciever.

Accounts

Accounts are fully handled for you! No need to worry about hosting your own servers

One Account To Rule Them All

Rotur uses a single account system, so you can use the same account on any game or operating system that uses Rotur, allowing for incredible interconnectivity

Rmail

Included with rotur and any account is a fully functioning emails system that works cross platform. Allowing you to get to working on the important stuff faster!

Rotur Credits

Handle user currency simply and easily, allowing transfers and an overarching rotur economy. Each user that logs in gets 1 credit per day.

Cloud Keys

Store and access key value pairs on the cloud in a user account. Allowing for cloud storage of files, game data, saves, and more!

Rotur Items

An online marketplace for items and data using rotur credits!
